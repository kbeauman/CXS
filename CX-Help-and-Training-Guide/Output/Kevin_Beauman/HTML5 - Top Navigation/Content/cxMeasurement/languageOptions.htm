Data Collection Information

Web-based Invitation, Tracker Window, Survey, Thank You Page, and Smart Thank You Pages

  • Invitation, Tracker Window, Survey, Thank You Page, and Smart Thank You Page Language - For additional investment, these cxMeasure items (all but IVR deployed) can be presented in any of the languages referenced at: http://www.junctioninternational.com/languages.html
  • Either a language qualifier page or a locale CPP can be used to determine survey language for a particular respondent.
  • Results collected from Non-English surveys are aggregated and presented in the English interface in the portal.
  • Open-ended questions are not translated.

Email Deployment

  • For client sent email deployment:
    • STANDARD: The client must include the relevant link within the invitation based on the language that the recipient needs.
    • CUSTOM: If the client does not know ahead of time which language needs to be presented to the respondent, then the email invitation must be dual/multi-language and a qualifier page must be built to allow the respondent to select which language version of the survey they want to access. A ticket should be submitted to “Product Requests 2” in the JIRA to allow for proper scoping, pricing, and approval.
  • If ForeSee is sending the invite:
    • STANDARD: A language ID variable must be included within the email address list that indicates which language version of the survey is to be linked within the invitation.
    • CUSTOM: If the client does not know ahead of time which language needs to be presented to the respondent, then the email invitation must be dual/multi-language and a qualifier page must be built to allow the respondent to select which language version of the survey they want to access. A ticket should be submitted to “Product Requests 2” in JIRA to allow for proper scoping, pricing, and approval.

IVR/IVA Deployment

  • Transcriptions of Non-English audio are available for nearly any language if we are partnering with SmartAction for the transcription service. Additional investment is required to accommodate. Consult with Sales Engineering.
  • IVA verses IVR - IVA is only available in English. IVR is available in any language that Voice Talent can be acquired (generally all the same languages we support in web-based survey translation). Our preferred vendor for voice talent is: http://www.walshmedia.com/green_room/. A list of languages supported can be found at this link.

SMS/Text Invitation

Text body must be in English, but can link to a Non-English survey.

Sweepstakes Incentives

If Non-English support required, custom incentive pricing must be provided from Offline Product Manager to accommodate different country rules and legalities.

ForeSee Feedback

English only, no timeline for non-English availability. See ForeSee Feedback.

Data Delivery Information

Portal

  • AAP is localized in English and French-Canadian (if other languages needed, consult Sales Engineering).
  • EP is localized in English, UK, Spanish, French-Canadian, and Portuguese (if other languages needed, consult Sales Engineering).
  • VOC Portal language options: TBD

Comment Cluster

Comment Cluster can be set to cluster based on language of the comments. The following languages are available: Arabic, Bulgarian, Czech, Chinese Simplified, Danish, Dutch, English, Estonian, Finnish, French, German, Greek, Hungarian, Italian, Irish, Korean, Latvian, Lithuanian, Maltese, Norwegian, Polish, Portuguese, Romanian, Russian, Slovak, Slovene, Spanish, Swedish, Thai, Turkish.

Sentiment Analytics

Phase 1, Commercial Release of Text Analytics will only be available for running on English comments. See Sentimental Analytics.

Automated Reporting/Scorecards

Only available in English.

SIR

Not available in Non-English unless done out of the Brazil office (Portuguese, Spanish).
