Alerts

Set up notifications for each project to be alerted to selected scenarios, such as a drop or increase in four-star ratings.

The Alerts tab can be displayed at any time from the toolbar. The page is comprised of a list of Alert Names and properties, along with the following functions:

  • Status - Toggles to turn the alert on or off.
  • - Edit this alert.
  • - Delete this alert.

You may also Add An Alert, which requires the same process as editing and is described in the following section.

To set up an alert:

The following process is for creating a new alert. However, the process is very similar for revising an existing alert.

During the Create process, the Warning icon () displays to indicate insufficient Trigger Events criteria and/or Publish information. The Valid icon () displays to indicate the alert is ready for use.
  1. On the Alerts page, click Add An Alert. For editing an existing alert, click the Alert Name or Edit icon ().
  2. Click the alert title and description to edit.
  3. In the Trigger Events container, select the desired Project Name. The Choose Criteria menu displays.
  4. Select the desired criteria.
    Star Rating = 5 Stars - An alert is triggered when a respondent enters an five star rating in the survey.
    Topic = Technical Issue - An alert is triggered when a respondent reports a possible problem with the web page.
  5. Optionally, you can click Add criteria to add another layer to the alert trigger.
    Topic = Technical Issue + Star Rating = 1 Star
  6. Click the Publish radio button. The Trigger Events container closes and the Publish container expands.
  7. Type the Subject and Email Body of the alert note. It is recommended to include some reference to the Alert Name and/or Trigger Events criteria to ensure the alert recipient understands the reason for the email.
  8. In the Email To text box, type the email address(es) to whom the alert must be sent. Separate multiple addresses with a comma.
  9. Click Save. The Alerts page displays with the new alert listed in active status.
