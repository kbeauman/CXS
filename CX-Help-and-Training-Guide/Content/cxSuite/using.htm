Using ForeSee CX Suite

Top Navigation Bar

The top navigation bar or toolbar, runs along the top edge of the page. This is the primary means of moving from one section or application of CX Suite to another. The ForeSee logo is a link to the CX 360 Dashboard. Next to the logo are the main application pages included in your subscription and are available for you to navigate directly to at any time.

Click the Help icon (?) to display this Help feature in a separate browser window. The last action feature on the toolbar is a list of clients you have access to, which is determined by your User Permissions, and Log Out.

Using Help

ForeSee Logo

Like the CX Suite toolbar, the ForeSee logo is a link to the Help Home page.

Search

In the upper-right corner of each Help page, a search text box displays where you can type in a key word or phrase and click the Search icon (). A page displays with each topic containing the key word(s).

Labels and Actions

As you view CX Suite Help you may notice items in bold and bold-italic. This is to help you identify Labels, such as a text field label, and Actions, as in a graphical user interface button or link. Other styles include:

  • Menu Options
  • System Output (or default settings)
  • Programming syntax
