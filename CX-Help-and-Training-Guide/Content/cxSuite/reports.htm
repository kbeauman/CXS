Reports

Manage report settings to download a Comma Separated Value (.csv) file of the data. This file type can be imported to another database or configured into a spreadsheet report, such as Microsoft Excel.

The Reports page can be displayed from the menu bar and offers the following options:

  • Report Types - These options based on your User Permissions which consists of On-Demand and Subscription report schedules.
  • Topic - Limit the report to data acquired from a single topic.

The right pane has two sections:

  • Create Report - Select the type of file to download, how the data should be selected, and the date range.
  • Settings - Select to have a mapping file of the survey included in the download.

To run a report:

  1. In the left pane, select the Report Type, Project, and Topic from the menus.
  2. In the right pane, if the Create Report section is collapsed, click the title to expand the section and view the options.
  3. Currently, there is only one option for Output and Selection. Click on the Date menu to display a calendar where you can click on the first and last day of the desired date range.
  4. Click Settings to expand the section and collapse the Create Report section.
  5. Click the Mapping File option and click Download. A dialog displays for you to select viewing or saving the file.
