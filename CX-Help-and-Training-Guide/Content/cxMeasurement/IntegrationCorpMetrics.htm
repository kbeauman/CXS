What Is It?

Integration - Corporate Metrics is a one-way import of summarized data and aggregate data above the respondent level that can be displayed in the ForeSee Portal for analysis and comparison purposes.

Types of information supported (list is not exhaustive):

  • Sales Volumes
  • Number of Visitors
  • Revenue Totals
  • Conversion Rates

Why Use It?

Forward thinking organizations need to see how customer experience impacts sales and ultimately shaping their business. ForeSee Integration - Corporate Metrics can load information from various business systems into ForeSee adding context for decision making and providing a comparative format for detailed analysis. Using ForeSee’s purpose built Integration for Summary Level information is fast, easy, and powerful.

General Information

  • Corporate Metrics cannot be set up for hierarchies.
  • Data only supported at the daily or monthly summary level.
  • Data must be passed via agreed upon file format (text delimited format will be presented to client as example).
  • Corporate Metrics is not part of standard integration and is a separate fee ($10K) per each separate data layout.
  • Corporate Metrics is not affected by and does not affect other integrations.
  • Implementation timeline requires prioritization among other projects in the queue. Work with CS and IS/Tech to identify.
  • If the client wants to increase the number of variables passing, they must provide the updates to the CSV file with an additional column and then pass it to us. Proper notification by all parties should be managed to ensure no failures in delivery or uploading process.
  • Updates on the client side should be done in an automated fashion.

Requirements For Passing Data To ForeSee

Corporate Metrics requirements are the same as the ones listed in the Respondent Level Integration section.

Corporate Metrics In Advanced Analytics Portal

Step 1 - Click on Sat Summary tab, then click on Corporate Metrics button.

  • The Corporate Metrics tab is only available within the upper-level Sat Summary tab, and is only visible for those clients who have it activated.
  • Corporate Metrics is activated at a client level, not a measure level. Once activated, any measure the client has can be compared to the Corporate Metrics.

Step 2 - Select Corporate Metrics filters.

  • Corporate Metrics can be either daily or monthly metrics. As such, you can select filters based on daily or monthly Corporate Metrics.
  • The delimited files which the client sends to us containing their Corporate Metrics must be separate for daily metrics and monthly metrics.

Step 3 - View your Corporate Metrics.

Based on the metrics you selected as filters in Step 2, your Corporate Metrics shows up as columns, along with a column for aggregate Satisfaction for that measure, for that day.

Corporate Metrics In Executive Portal

Step 1 - Select ‘Score’ and then ‘Trend Correlation’.

Step 2 - Select what you want to compare Corporate Metrics against as trend lines for ‘Selection #1’, and then select Corporate Metrics for ‘Selection #2’.
