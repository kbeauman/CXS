Using ForeSee® CX Suite

Navigation

Top Navigation Bar

The top navigation bar, which runs along the top edge of the page, is the primary means of moving from one page to another. The ForeSee® logo is an action feature that you can click on to return to the Projects page. Next to the logo, the main application pages are available for you to navigate directly to at any time.

Click the Help icon (?) to display this Help feature in a separate browser window. The last action feature on the toolbar is your user name and a menu of features, such as Client Search, a list of clients, and Log Out, which is determined by your User Permissions.

Projects Page

The Projects page displays a list of projects you have access to, which are grouped by application, such as ForeSee® Feedback and ForeSee® Replay. Each project name can be clicked to edit. There are also icons on the right to navigate to a specific page within the project. A tooltip displays when you mouse over the icon to inform you what the icon is for.

Using Help

Search

In the upper-right corner of each Help page, a search text box displays where you can type in a key word or phrase and click the Search icon (). A page displays with each topic containing the key word(s).

Labels and Actions

As you view CX Suite Help you may notice items in bold and bold-italic. This is to help you identify Labels, such as a text field label, and Actions, as in a graphical user interface button or link. Other styles include:

  • Menu Options
  • System Output (or default settings)
  • Programming syntax
