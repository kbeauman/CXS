Filters

Pages displaying a sidebar on the left offer a variety of ways in which data can be filtered. The sidebar's width may be adjusted by dragging the grab handle () along the right edge. The entire bar can be collapsed by clicking the toggle icon ().

Common filtering options include:

Date

This is the date range from which the displayed data comes from. You can revise this range in the following manner:

  1. Click the current date range to display a calendar page with the current range highlighted.
  2. Click the beginning and end dates to highlight the desired date range (using the scroll icons on either side of the month names if necessary).
  3. Click Apply. The calendar page closes and the screen refreshes to display data for the new date range.

Project

The Project menu displays the Project Name for the displayed data. Click the Project Name to open the menu and select another project to view.
