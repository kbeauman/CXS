Custom Data Feeds

You can upload data from outside sources, such as social media, for processing by Text Analytics. Additionally, you can set Text Analytics to process data automatically through the ForeSee® API. Data files must meet the following requirements:

  • Comma Separated Values (CSV) format
  • Maximum 1000 rows per file
  • Column headers must say Text or Timestamp
  • Text columns format must be text, with 15-10,000 characters per comment
  • Timestamp columns must be ISO format of YYYY-MM-DDThh:mm:ss (empty columns are filled automatically with the UTC Date and Time of the upload)

Upload Data

  1. Go to Global Settings > Apps > Text Analytics > Custom Data Feeds. The Custom Data Feeds page displays.
  2. Click Create New Feed. A three-step wizard displays.
  3. Name Your New Feed - In the Name text box, type a required label that is easily associated with the type of data in the file. Next becomes enabled. Optionally, type any additional information in the Description text box and click Next.
  4. Choose File Type - Click Select this Option for the desired type of data feed:
    • Upload a CVS - The Process File page displays where you can type the file directory, or browse to the desired CVS file.
    • Generate Feed ID (API) -
  5. Click Finish. The file is first validated and then processed.
