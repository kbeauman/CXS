Users & Roles

This section of Global Settings is for Site Administrators only and is designed to provide control over each user's access and permissions to the various CX Suite features and functions.

Your organization has hired a new analyst to review and report on your Feedback data. Subsequently, you need to perform the following tasks:
  1. Add the new user.
  2. Create a User Role with access to Feedback Respondent Data only.
  3. Assign the user to the Role.

Users & Roles offers the following tabs:

  • Users
  • Unmatched Users
  • Organizational Users
  • Roles
  • Version History
