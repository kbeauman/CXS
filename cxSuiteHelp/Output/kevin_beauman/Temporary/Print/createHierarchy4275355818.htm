Create a Hierarchy

  1. On the Hierarchy Management page, click Set Up a New Hierarchy. The Set Up Hierarchy page displays with the initial steps for creating a new hierarchy version along with a hierarchy example and template that you can download.
  2. In Hierarchy Name, type a descriptive label for the new version that will help identify the hierarchy's purpose and application.
  3. Click the Hierarchy Type:
    • Location - Intended for retail, franchise, or distribution type organizations using store measures, such as Point of Sale or Loyalty Program. An example of a Location Hierarchy is: Region > District > Store.
    • Employees - Intended for organizations using the Employee Satisfaction measure. An example of an Employee Hierarchy is: COs > Managers > Team Leads > Laborers.
    • Agents - Intended for organizations using Contact Center measure or Case Management. An example of an Agents Hierarchy is: Zone > Call Center > Operator.
    The selection is enabled and displays a green check mark.
    If you have an existing hierarchy, even in Draft, the corresponding Hierarchy Type does not display.
  4. Click Browse Files to select and upload a Comma Separated Values (CSV) file. Next becomes enabled. If you need to prepare a CSV file, click either Hierarchy Only or Hierarchy + Master Data Attributes to download a template. See Create a Hierarchy CSV File for instructions.
  5. Click Next. A wizard displays when processing of the file is complete.
    If you find there is a problem with your CSV file and need to make revisions, a link is available along the bottom of the wizard to upload a new file.
  6. Key Selection - Each value in the top row of your CSV file is available in the Column menus of the following keys. Currently, spaces are not allowed in column headings, so you may choose to revise the Display Name of the selected column or accept the default value. Data Preview displays the first few rows of data in the selected column as a reference. See Hierarchy Attributes for detailed descriptions of these settings.
  7. Master Data Setup
