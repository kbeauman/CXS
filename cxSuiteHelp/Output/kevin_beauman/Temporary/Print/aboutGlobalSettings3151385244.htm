About Global Settings

A Global Settings feature is available from the User Menu to provide quick access to your controls and the latest version of ForeSee’s Hosted Code. Access to Global Settings is limited and managed by ForeSee Support.

When you click Global Settings on the User Menu, a separate section displays with your CX Suite applications on the left with links to the available settings listed below each application. Click Integrate to access the Hosted Code Publishing page, which offers the following settings:

  • Code snippet for your templates
  • Hosted Code version settings
  • On/Off settings for your CX Suite applications
  • User tracking options
